# OOP.py

👨‍💼 Employee Management System Using OOP

📖 Project Overview

The Employee Management System Using OOP is a beginner-friendly Python project developed to demonstrate the core concepts of Object-Oriented Programming (OOP). The system allows users to manage employee records through a simple menu-driven interface.

Users can add employees, managers, and developers, search employee records, update salaries, display all records, and delete employees. The project also demonstrates inheritance, encapsulation, constructors, destructors, method overriding, and polymorphism.

⸻

🎯 Objectives

* Understand Object-Oriented Programming concepts.
* Implement inheritance and encapsulation.
* Perform CRUD (Create, Read, Update, Delete) operations.
* Build a menu-driven console application.
* Practice working with classes and objects.

⸻

✨ Features

* ➕ Add Employee
* 👔 Add Manager
* 👨‍💻 Add Developer
* 📋 Display All Employees
* 🔍 Search Employee by ID
* 💰 Update Employee Salary
* 🗑️ Delete Employee
* 🚪 Exit the Program

⸻

🛠️ OOP Concepts Used

* ✅ Class and Object
* ✅ Constructor (__init__)
* ✅ Destructor (__del__)
* ✅ Inheritance
* ✅ Encapsulation (Private Variables)
* ✅ Getter and Setter Methods
* ✅ Method Overriding
* ✅ Polymorphism
* ✅ super() Function
* ✅ issubclass() Function

⸻

📂 Classes

1. Employee

Stores employee details such as Employee ID, Name, Age, and Salary.

2. Manager

Inherits from the Employee class and includes the Department attribute.

3. Developer

Inherits from the Employee class and includes the Programming Language attribute.
